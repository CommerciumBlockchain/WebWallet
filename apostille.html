COMMERCIUM APOSTILLE

COMMERCIUM APOSTILLES are servers that run a Commercium wallet and make decisions. COMMERCIUM APOSTILLE are required to have 10,000 CMM collateral, a dedicated VPS (IPv4 address, 1GB RAM, 20GB drive) and be able to run 24 hours a day without a more than an hour of connection loss. COMMERCIUM APOSTILLE get paid 45% of the block reward on every block, which is distributed to COMMERCIUM APOSTILLE one at a time.

HOW MUCH CAN I EARN FROM HOSTING A COMMERCIUM APOSTILLE?

45% Block Reward paid to the node waiting the longest to be paid.

WHAT DO I NEED TO HOST A COMMERCIUM APOSTILLE?

  • 20,000 CMM: Arguably the hardest part. CMM can be obtained from exchanges such as CryptoBridge, Crex24.
  • A computer for running a local wallet to start nodes and hold funds
  • A VPS server installed Ubuntu for remote node running 24hrs/day with the following specs:
    • 1GB RAM (about half used for OS and half for daemon)
    • 20GB drive (less than 1GB used for wallet, some for OS, and some for future.)
    • 1 Static IPV4 Address per node (IPV4 address that never changes) (No IPV6)
  • A little time: Pick your time and follow this Setup Guide. There are a few guides that I hope will make the process a lot easier. Video-Tutorial: How to set up a Commercium Apostille
